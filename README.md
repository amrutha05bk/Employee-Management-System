Employee Management System (EMS)
A 3-day Core Java project demonstrating fundamental and advanced Java concepts.

Technologies Used
Java 17
MySQL 8.0
JDBC
Eclipse IDE

Project Structure
Day 1 - Core Java (Inheritance, Interfaces, Abstract Classes)
Day 2 - Collections, Streams, Functional Interfaces
Day 3 - JDBC, Service Layer, MySQL Database

Packages
Package	Description
com.ems.model	Core Java model classes
com.ems.collections	Collections demo
com.ems.streams	Streams and functional interfaces
com.ems.util	Database utility
com.ems.service	CRUD service classes

How to Run
Import project into Eclipse
Add mysql-connector-java.jar to build path
Run SQL script to create emsDB database
Update password in DBUtil.java
Run MainApp.java

Features
Employee CRUD operations
Department management
Project management
Grade calculation based on salary
Input validation
JOIN queries
